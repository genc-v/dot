# My Tools

Here is a list of tools I use across my Linux setups:

## CLI Tools
- `neovim` (Text Editor)
- `tmux` (Terminal Multiplexer)
- `fish` (Shell)
- `zoxide` (Smarter cd)
- `lazydocker` (Docker TUI)
- `lazygit` (Git TUI)
- `fzf` (Fuzzy finder)
- `opencode` (AI CLI Agent)

## GUI / Desktop
- `hyprland` (Window Manager)
- `kitty` / `alacritty` (Terminal Emulator)
